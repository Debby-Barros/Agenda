System Metaphor

Web Browser: Dispositivo para armazenar as informações do front-end que utilizam as tecnologias HTML, CSS e JavaScript e tem a capacidade de fornecer a interface para o usuário.
Servidor X: Dispositivo para armazenar as informações do back-end que utilizam a tecnologia Flask que é um mini framework escrito em python usado para desenvolvimento web.
Servidor Y: Dispositivo para armazenar os dados que utilizam o SGBD MySQL que cria e gerencia o banco de dados que será utilizado em conjunto com o Flask.
